    CSS grid

    Css grid er blevet omtalt som det største inden for CSS i nyere tid. Før grid var det nødvendigt at bruge ting som tabels, floats og posistioning for sit layout, men med CSS grids row og collumn system kan man simpelt placere elementer hvor man vil i layoutet.

    For at lave et grid skal man først have en parent container med selectoren display grid, derefter skal man bestemme hvor mange rows og collumns ens layout skal bestå.

    Man kan bruge forskellige metoder til at placere sine elementer i sit grid system. Man kan bruge span som virker ved at bestemme i hvilke collumn og row elementet skal starte og hvor det skal ende.

    Men nok den bedste metode er Grid-template-areas som også arbejder med columns og rows men i stedet for at skrive start og slutpunkt placere man elementer ved at navngive grid placeringer.
