    Hyper text Markup Language

    Html står for Hyper Text Markup Language og er det grundlæggende sprog man bruger til at bygge hjemmesider op med. Man kan sige at html er til en hjemmeside, hvad en stål konstruktion er til et højhus.

    Form elementes

    I vores html og CSS workshop grupper havde min gruppe om html form elements og form styling og da form elements findes overalt på nettet synes jeg de er vigtige at forstå.

  • Attributes rolle er at give mere information om et html element.
  • Alle elementer kan have attributes.
  • Attributes består for det meste af 2 dele. (Name og value)
  • id og class bruges som et link til et stylesheet.
  • Form elements bliver brrugt til at indsamle bruger data.
  • Den vigigste er nok input, og kan bruges til at lave et tekstfelt, button osv.
  • Select elementet skaber en dropdown menu.
